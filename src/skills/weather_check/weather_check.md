This skill provides current weather information for a specified location. It fetches data from a weather API.

### When to Use
Use this skill when the user asks for current weather conditions in a specific city or region.

### Inputs
- `location`: The name of the city or region (e.g., 'Moscow').

### Outputs
Returns a string with the current weather conditions, temperature, and other relevant details.

### Example
```python
weather_check("Moscow")
```
