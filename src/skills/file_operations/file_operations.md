Этот навык предоставляет инструменты для выполнения операций с файлами, включая чтение, запись, поиск и удаление текстовых файлов. Он использует стандартные библиотеки Python (`os`, `pathlib`) для работы с файловой системой.

**Когда использовать**: Когда пользователю нужно прочитать содержимое файла, записать данные в файл, найти строки в файле или удалить файл.

**Входы/Выходы**:
  - Чтение файла: путь к файлу → содержимое файла.
  - Запись в файл: путь к файлу, текст → подтверждение записи.
  - Поиск в файле: путь к файлу, строка для поиска → список строк, содержащих искомую строку.
  - Удаление файла: путь к файлу → подтверждение удаления.
