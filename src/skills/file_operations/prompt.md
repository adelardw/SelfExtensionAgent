Используй этот навык для работы с текстовыми файлами. Примеры вызовов:
1. Чтение файла: `read_file("path/to/file.txt")` → возвращает содержимое файла.
2. Запись в файл: `write_file("path/to/file.txt", "текст для записи")` → подтверждает запись.
3. Поиск в файле: `search_in_file("path/to/file.txt", "искомая строка")` → возвращает строки, содержащие искомую строку.
4. Удаление файла: `delete_file("path/to/file.txt")` → подтверждает удаление.

Все операции выполняются с обработкой ошибок (например, если файл не существует).
