Устанавливает Python-пакеты через `uv` (ультрабыстрый менеджер пакетов, альтернатива `pip`).
**Когда использовать**:
- Когда требуется установить пакет в текущее окружение Python.
- Когда нужна максимальная скорость установки (особенно для больших проектов).
**Входы**:
- `package_name`: Имя пакета для установки (например, `requests`).
- `version`: Опциональная версия пакета (например, `==2.31.0`).
**Выходы**:
- Сообщение об успешной установке или ошибке.
