The `file_organizer` skill organizes files in a directory by their extensions. Use this skill when you need to declutter a directory with mixed file types.

**How to use:**
1. Provide the `directory_path` where the files are located.
2. The skill will create subdirectories for each file type and move the files accordingly.

**Example call:**
```python
file_organizer("/path/to/directory")
```
