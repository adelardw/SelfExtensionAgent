This skill helps organize files in a specified directory by moving them into subdirectories based on their file extensions. It is useful for decluttering directories with mixed file types.

**When to use:**
- When a directory contains many files of different types (e.g., images, documents, videos).
- When you want to automate the organization of files.

**Inputs:**
- `directory_path`: The path to the directory to organize.

**Outputs:**
- A confirmation message listing the number of files moved and their new locations.

**Example:**
```python
file_organizer("/path/to/cluttered/directory")
```
