This skill provides tools for working with text files: reading, writing and searching content.

**When to use:**
- When user needs to read content from a text file
- When user needs to write text to a file
- When user needs to search for specific text within files

**Inputs/Outputs:**
- Input: File path and optional text content
- Output: File content or operation confirmation

**Examples:**
- "Read the contents of notes.txt"
- "Save this text to reminder.txt"
- "Search for 'important' in all .txt files in docs folder"
